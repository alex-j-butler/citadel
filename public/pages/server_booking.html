Booking and Setup Server Guide

This will outline the process of booking a server for your games. Please be aware we are only supporting configs for Ready Steady Pan, ozfortress and our own league configs for Highlander. Furthermore, including config support we will also be supporting the respective maps they use in each of their seasons.

Booking

  1. Click here to join the Discord server
  2. Click on the #booking channel
  3. Type book or /book in the text channel to initiate the booking process
  4. You are done, the bot will send you a private message on Discord with all the details required to use your server. Remember to ensure at least two players are present on the server, otherwise it will unbook in 15 minutes.

Type help in the #booking channel if you need assistance on booking commands 

book 				- Book a new server
unbook				- Unbook your current server
send password			- Send the updated server details
demos				- Send the link to the uploaded demos
help				- Display the help message

For help, contact staff or post in the #support channel
Note: ozfortress booking commands also are accepted by this bot

Retrieving STV demos

Once the server you have booked has either expired or been manually unbooked the STV demo will be messaged in the "Booking" channel, being targetted at the person who originally booked the server. The STV should upload nearly instantaneously, if the link is not sent after 1 hour of the server being unbooked, contact an admin.

Setup

The following will show what configs, exec aliases and maps we support. It will also go into setting up a server for the various formats.

Before going forward please make sure that you have the rcon password and enter the following into your console, rcon_password (password here).

Maps

Maps that are present on the bookable servers and fastdownload will only contain such maps that the leagues we support are and our own format. These will include such map types as cp, koth and payload.

When changing the level of a server, use the following command rcon changelevel (map here).

Aliases

Map aliases are a shortened way to change your maps on the server. Instead of typing rcon changelevel koth_product_rc8 you can now type rcon product. Some important information to note is map aliases are primarily set for specific maps that are used within our supported league configs. Newer versions of maps must be executed manually until those maps are eventually introduced for play.

Configs are files that have commands that are run which edit numerous variables on how the game is played on the server.

When executing a config for setup on a server, use the following command rcon exec (config here). For supported configs, see the reference below.
